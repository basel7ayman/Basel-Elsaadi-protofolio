## 🔍 Overview

This is a single‑page portfolio website built with **vanilla HTML & CSS** and a focus on:

- Clean, modern **dark UI**
- Clear structure for **About, Education, Experience, Projects, and Activities**
- Easy navigation with a **sticky navbar** and smooth scrolling
- Direct links to my **GitHub profile**, **resume**, and **live project demos**
